# AFL Phase 5 — Live Unreal Hook

## Purpose

Introduce a live file-watch hook so Unreal-emitted event logs can be consumed automatically by AFL runtime.

## Active transport

- file watch for live demo iteration

## Future transport

- socket/live stream using the same event schema

## Outputs

- live_match_result.json
- live_reward_result.json
- live_event_timeline.json
