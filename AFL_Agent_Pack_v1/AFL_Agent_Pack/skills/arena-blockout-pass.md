# Arena Blockout Pass

## Purpose
Build a readable map shell before visual polish.

## Procedure
1. Establish footprint and lane widths with primitive geometry.
2. Mark deployment bays, center lane, flank lanes, midfield hotspot, objective core.
3. Test robot capsule width and turning radius.
4. Add only major blockers and landmark placeholders.
5. Capture top-down and chase-cam review shots.

## Deliverables
- blockout map
- lane dimension notes
- first-pass screenshots

## Validation
- center lane supports direct pressure
- both flanks are traversable
- objective is visible from intended approaches
- no decorative clutter is added early
